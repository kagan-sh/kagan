# ᘚᘛ KAGAN

AI-powered Kanban TUI for autonomous development workflows.

## Quick start

```bash
uv run kagan
```

## Start here

- Read the [User Guide](user-guide.md) for a full walkthrough of workflows and automation.
- Check [Configuration](config.md) for agent setup and scheduler options.

## Key bindings

| Key | Action |
| --- | ------ |
| <kbd>j</kbd>/<kbd>k</kbd> | Navigate up/down |
| <kbd>h</kbd>/<kbd>l</kbd> | Navigate left/right |
| <kbd>n</kbd> | New ticket |
| <kbd>e</kbd> | Edit ticket |
| <kbd>d</kbd> | Delete ticket |
| <kbd>[</kbd>/<kbd>]</kbd> | Move ticket backward/forward |
| <kbd>Enter</kbd> | View details |
| <kbd>o</kbd> | Agent output |
| <kbd>s</kbd> | Start agent |
| <kbd>x</kbd> | Stop agent |
| <kbd>c</kbd> | Planner chat |
| <kbd>?</kbd> | Command palette |
| <kbd>q</kbd> | Quit |

## What it is

Kagan is a Textual-based Kanban board for coordinating autonomous development work. It keeps local state, supports agent-driven execution, and focuses on fast keyboard workflows.
