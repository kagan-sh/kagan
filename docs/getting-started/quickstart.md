# 5-Minute Quickstart

## TL;DR

```bash
uv tool install kagan
cd your-project
kagan
```

## 1. Launch

Run inside your repository:

```bash
kagan
```

If prompted, pick or create a project and active repo.

## 2. Create a task

Press `n`, then enter:

- Title
- Description
- Acceptance criteria
- Task mode (`AUTO` or `PAIR`)

Save with `F2` (or `Alt+S`).

## 3. Run work

### AUTO

Press `Enter` or `a` to start. Open **Task Output** with `Enter` to monitor progress.

### PAIR

Press `Enter` to open an interactive session (tmux / VS Code / Cursor).

Need help choosing? See [AUTO vs PAIR](../how-to/task-modes.md).

## 4. Review and finish

1. Move to `REVIEW`
1. Open **Task Output** with `Enter`
1. Use Summary/Diff/Review Output/Agent Output tabs
1. Approve or reject
1. Merge when ready

## 5. Learn fast keys

- Help: `?` or `F1`
- Actions palette: `.` or `Ctrl+P`
- Settings: `,`
- Debug log: `F12`
